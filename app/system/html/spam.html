Настройка анти-спама

В ответ на растущий спам в сетяз, ThinkSAAS создала альянс против спама в контенте, который будет бороться со спамом со дня его запуска.

Блокировка пользователя: Пользователям запрещается одновременно вход в систему по логину/нику и по IP-адресу.
Метод работы: пользователь -> блокировка
Управление пользователями >>
Очистить данные пользователя: Очистить весь спам, отправленный пользователями.
Метод работы: пользователи -> очистить данные
Управление пользователями >>
Фильтрация спам-слов: Фильтруйте и устраняйте спам в контенте, включив «плагин фильтра слов (wast_word)»
Метод работы: плагины -> плагин фильтра слов (wast_word)-> включить
Управление плагинами >> | Обновление облачного фильтра >>
Код подтверждения: Осложните спамерам жизнь, включив коды подтверждения при публикации и комментариях.
